## Usage

1. Open the application at [https://shklala.github.io/feedback-creator/](https://shklala.github.io/feedback-creator/)
2. Select an **Issue Category** from the dropdown
3. Choose the specific **Issue Type** (auto-populated based on category)
4. Select the **Session Slot** time
5. Enter the **Student ID**
6. Add optional **Notes to Moderation**
7. Review/edit the auto-filled **Issue Description**
8. Click **Generate** to create the formatted report
9. Click **Copy** to copy the report to your clipboard

## Issue Categories

- **Technical & Joining Issues** - Audio/video problems, connection issues, Zoom failures
- **Student Behavior & Engagement Issues** - Disruptive behavior, unresponsiveness, distractions
- **Session Logistics & Attendance** - Parent attendance, student wellness, session freezing
- **Tutor & Platform Account Issues** - Login problems, access denied errors
- **Minecraft & Software Issues** - Game login failures
- **Additional Problems** - Miscellaneous session issues

## Report Format

The generated report includes:
- Issue category (uppercase)
- Issue type
- Moderator mention (@ModeratorName)
- Student ID
- Session slot time
- Detailed issue description
- Optional notes
- Closing message
