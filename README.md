## Usage

1. Open `index.html` in your web browser
2. Fill in the form fields:
   - Student Name
   - Student Code
   - Date
   - Grade
   - Issue Description
3. Click "Generate" to create the report
4. Click "Copy" to copy the generated report to your clipboard
